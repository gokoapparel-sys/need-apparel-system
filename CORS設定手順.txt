================================================================
Firebase Storage CORS設定手順（最も簡単な方法）
================================================================

【重要】PDFに画像を表示するには、この設定が必須です

----------------------------------------------------------------
手順1: Google Cloud Consoleにアクセス
----------------------------------------------------------------
1. ブラウザで https://console.cloud.google.com/ を開く
2. 右上のプロジェクト選択で「goko-apparel-system」を選択

----------------------------------------------------------------
手順2: Cloud Storageを開く
----------------------------------------------------------------
1. 左側のメニュー（☰）をクリック
2. 「Cloud Storage」→「バケット」を選択
3. 「goko-apparel-system.firebasestorage.app」をクリック

----------------------------------------------------------------
手順3: バケットの詳細ページ
----------------------------------------------------------------
1. 「構成」タブをクリック
2. 「CORS」という項目を探す
3. 「編集」ボタンをクリック

----------------------------------------------------------------
手順4: CORS設定を追加
----------------------------------------------------------------
以下のJSON設定をコピー＆ペーストしてください：

[
  {
    "origin": ["*"],
    "method": ["GET", "HEAD"],
    "responseHeader": ["Content-Type", "Access-Control-Allow-Origin"],
    "maxAgeSeconds": 3600
  }
]

----------------------------------------------------------------
手順5: 保存して確認
----------------------------------------------------------------
1. 「保存」ボタンをクリック
2. ブラウザに戻って、完全リロード（Ctrl + Shift + R）
3. PDF出力を再実行
4. PDFに画像が表示されることを確認

================================================================
設定完了後の確認
================================================================
✓ ブラウザのコンソールでCORSエラーが消えている
✓ PDFに実際の画像が表示される（プレースホルダーではない）

================================================================
トラブルシューティング
================================================================
Q: 「構成」タブが見つからない
A: バケット名を正しくクリックしているか確認してください

Q: 設定したのに画像が表示されない
A: ブラウザのキャッシュをクリアして、シークレットモードで試してください

Q: エラーメッセージが表示される
A: JSON形式が正しいか確認してください（カンマ、括弧など）
