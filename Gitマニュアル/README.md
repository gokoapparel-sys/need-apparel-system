# Gitマニュアル

このフォルダには、GOKO アパレル商品管理システムのGit作業マニュアルが含まれています。

## 📁 ファイル一覧

### Git作業マニュアル.md
- Markdown形式のマニュアル
- テキストエディタ（VSCode、メモ帳等）で閲覧可能
- Gitで管理される

### Git作業マニュアル.html
- HTML形式のマニュアル
- ブラウザで閲覧可能
- 印刷に最適化されたデザイン
- カラフルで読みやすい

## 📖 マニュアルの内容

1. **このPCで作業をする場合** - 通常の作業開始・終了手順
2. **別なPCで初めて作業をする場合** - 初回セットアップ手順
3. **別なPCで作業を再開する場合** - 2回目以降の作業手順
4. **トラブルシューティング** - よくあるエラーと解決策
5. **よく使うGitコマンド一覧** - コマンドリファレンス

## 🌐 HTMLマニュアルを開く

エクスプローラーで `Git作業マニュアル.html` をダブルクリックするか、以下のコマンドを実行：

```powershell
start "C:\Users\aja10\Desktop\goko_appalel-system\Gitマニュアル\Git作業マニュアル.html"
```
